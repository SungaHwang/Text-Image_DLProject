단순히 html css js를 이용하여
무신사 웹사이트의 프론트 부분만을 따라 구현했습니다

### 간단한 기능
---

- 랭킹 차트 순차적으로 보여주기

  슬릭 슬라이더 라이브러리를 이용

- 품목 세부 메뉴 더보기

- 품목 부분 토글링

- 네비게이션 메뉴 마우스오버 시 하위 메뉴 노출
